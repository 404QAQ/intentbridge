# Web UI Screenshots

This directory should contain screenshots of the IntentBridge Web UI.

## Required Screenshots

Please capture the following screenshots and save them in this directory:

### 1. Dashboard
- **dashboard-light.png** - Dashboard in light mode
- **dashboard-dark.png** - Dashboard in dark mode

### 2. Requirements List
- **requirements-list-light.png** - Requirements list in light mode
- **requirements-list-dark.png** - Requirements list in dark mode
- **requirements-filters.png** - Requirements list with filter panel open
- **export-dropdown.png** - Export dropdown menu

### 3. Theme Toggle
- **theme-toggle.png** - Theme toggle button in header

## How to Capture Screenshots

1. Start the Web UI:
   ```bash
   ib web start
   ```

2. Open in browser: http://localhost:3000

3. Use browser's built-in screenshot tool or:
   - macOS: Cmd + Shift + 4, then Space, click window
   - Or use browser DevTools: Cmd + Shift + P, type "screenshot"

4. Save screenshots to this directory with the filenames listed above

## Screenshot Guidelines

- Resolution: 1920x1080 or higher
- Format: PNG
- Content: Ensure no sensitive data is visible
- Quality: Use full quality, no compression
